README FILE

The file "DSA_assignment_group_28.c" contains the implmentation of an Rtree using the STR approach.
In order to run the file, please enter the following commands:
gcc -c rtree.c
gcc -o rtree_exe rtree.o

After making the executable please input your file name when running the executable. It should be entered in the following format.
./rtree_exe <file_name>

If your input file is data.txt, please run it as shown below:
./rtree_exe data.txt
